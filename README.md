# Alex B Engineering Portfolio

This is a simple static portfolio website built with HTML, CSS, and JavaScript. It is designed to be hosted using GitHub Pages.

## Files

- `index.html` — main website content
- `style.css` — styling and responsive layout
- `script.js` — small interaction for project cards
- `.nojekyll` — tells GitHub Pages not to process the site with Jekyll
- `assets/docs/resume.pdf` — put your resume PDF here and name it `resume.pdf`
- `assets/img/` — put project photos, CAD renders, PCB screenshots, or diagrams here

## What to edit first

1. Replace `your-email@example.com` in `index.html` with your real email.
2. Replace the GitHub and LinkedIn links near the bottom of `index.html`.
3. Add your resume PDF to `assets/docs/` and name it `resume.pdf`.
4. Add project images to `assets/img/` and replace the placeholder blocks in `index.html` with image tags.

Example image replacement:

```html
<img class="project-image" src="assets/img/can-bus-board.jpg" alt="CAN bus monitoring PCB" />
```

## GitHub Pages upload steps

1. Make or sign into a GitHub account.
2. Create a new repository named exactly `YOUR-GITHUB-USERNAME.github.io`.
3. Upload all files from this folder into the repository root.
4. Go to the repository settings.
5. Click **Pages** on the left sidebar.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select the `main` branch and `/root` folder, then save.
8. Visit `https://YOUR-GITHUB-USERNAME.github.io` after GitHub finishes deploying.

## Important privacy note

Do not upload confidential nuclear plant information, internal documents, restricted drawings, security details, or employer-protected files. Keep project descriptions high-level when needed.
