# Alexandrae Britos Engineering Portfolio

Static engineering portfolio built with HTML, CSS, and JavaScript for GitHub Pages.

## Files

- `index.html` — main portfolio page
- `style.css` — styling
- `script.js` — project card toggle behavior
- `assets/docs/resume.pdf` — resume linked by the View Resume buttons
- `assets/img/` — add future project images here

## What still needs to be customized

1. Replace the LinkedIn placeholder in `index.html` with your real LinkedIn profile URL.
2. Add project photos to `assets/img/`.
3. Replace the project placeholder boxes with real images once available.
4. Keep confidential nuclear/plant information, internal drawings, and protected photos out of the public site.

## GitHub Pages

Upload these files to the root of your GitHub Pages repository and deploy from the `main` branch using `/ (root)`.
